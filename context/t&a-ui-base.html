Totumas & Aventuras

"Every quest is a mirror. Every monster is a wall. Two heroes, one vessel."

1. The Visual Philosophy

The interface must feel **forged, not rendered**. We avoid rounded pills and bright pastels. We use shadows to suggest that the app is a physical space—a sanctuary for two players. The Totuma (vessel) metaphor dictates that information is "contained" and shared, never scattered.

2. Narrative UI Logic

Epic Context

Use Cinzel for titles to evoke the weight of history and myth. High contrast between headers and parchment-colored text creates a sense of legibility in the dark.

Intimate Context

Use Lora for narrative text. Its italics feel like handwritten letters, grounding the "adventure" in the reality of the two people sitting together.

/* Implementation Note */ h1 { font-family: 'Cinzel'; font-weight: 700; } body { font-family: 'Lora'; font-weight: 400; }

The Core Loop

"Does this action bring us closer?"

Every button click in Totumas & Aventuras should require an intentional choice. We avoid "Ghost Buttons." Every choice has weight. Every choice is visible to the other hero.
